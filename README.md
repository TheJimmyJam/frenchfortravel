# The Cunning Linguist

A self-paced language learning platform. First course: 30-day French-for-travel for Paris confidence. Multi-language support coming next.

**Live app:** https://cunninglinguist.netlify.app

## What's in here

- **`index.html`** — the interactive web app. 30 daily lessons (350 vocab + 77 phrases), flashcards with spaced repetition, multiple-choice quiz, 8 conversation scenarios, progress tracking. Single file, no build step. Saves progress in your browser via localStorage.
- **`30-Day-Plan.md`** — the structured curriculum overview.
- **`Conversation-Scenarios.md`** — 10 real Paris dialogues (café, hotel, métro, etc.).
- **`Vocab-Tracker.xlsx`** — every word and phrase across 4 sheets, with status dropdowns and progress formulas.

## Daily flow

1. Open the app.
2. Read today's lesson aloud (5–10 min).
3. Drill flashcards (5–10 min).
4. Take the quiz (5 min).
5. Review one conversation scenario (5 min).
6. Mark the day complete to advance.

## Deployment

Netlify deploys `index.html` on every push to `main`. No build step.
